STEPS FOR RUNNING INTELLIGENT BUS STOP RECOGNITION SYSTEMS

1. Install Python 2.7 depending on your Operating System, via Continuum Anaconda (https://www.continuum.io/downloads)

2. Open "cmd" or "Power Shell" in Windows, or Open a Terminal in Linux.

3. In the terminal, type "python file_name.py" and press Enter.

4. Repeat the above step for each of the three files (three approaches).
